git status - ca sa vezi ce modificare ai facut si in ce fisier
git add . - ca sa adaugi toate fisierele pentru commit
git commit -m "nume la commit" - faci commit ca sa poti sa le pui pe repositoriu
git push origin master - faci push la cod pe github

git checkout .  - stergi totate modificarile tale
git dif - face difrenta intre local si ce e pe master (server/remote)

git pull origin master - iei code de pe github
